# Project coreDVD

это проект с реальными прошивками с собственным ассемблером и процессорами.

структура репозитория:

```bash
project-coredvd
├── LICENSE
├── README.md
├── asm.py
├── convert.py
├── docs.md
├── firmware
│   ├── Pulse
│   │   ├── docs.md
│   │   └── firmware.asm
│   └── z80
│       ├── docs.md
│       └── firmware.asm
└── src
    └── main.cpp
```

мы назвали наш процессор pulse.
